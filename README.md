# Todo App with Security

Une application Todo sécurisée avec Spring Boot et React.

## Structure du Projet

- `todo-backend/` : API REST Spring Boot
- `todo-frontend/` : Application React

## Prérequis

- Java 17+
- Node.js 16+
- Maven
- MySQL/PostgreSQL

## Installation

### Backend

```bash
cd todo-backend
mvn clean install
